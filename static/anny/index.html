Installation

Install using npm:

npm install happy-2-years-felix --save

Usage

This package exports a React component that when mounted will start to add "Happy 2 years Felix!" in random colors at random placements in random time intervals.

import React from 'react'; import { FelixAnniversary } from 'felix-2-year-anniversary'; function App() { return ( <div className="App"> <h1>My Application</h1> <FelixAnniversary /> <p>Other content goes here...</p> </div> ); }

Props

Prop Type Default Description
maxMessages number 50 Maximum number of messages to display before older ones are removed
minInterval number 0 Minimum time interval in milliseconds
maxInterval number 5000 Maximum time interval in milliseconds
customText string "Happy 2 year anniversary Felix!" Custom message to display
